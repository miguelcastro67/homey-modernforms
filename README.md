# Modern Forms for Homey

Bring your Modern Forms smart ceiling fans into Homey with fast, reliable local control.

This Homey SDK v3 app automatically discovers compatible Modern Forms fans on your local network and provides complete integration with Homey, including fan speed, lighting, fan direction, Breeze Mode, and Advanced Flow support.

## Features

- Automatic discovery of compatible fans
- Local network communication (no cloud required)
- Fan On/Off
- Six fan speeds
- Fan direction
- Breeze Mode
- Light On/Off
- Light brightness control
- Automatic synchronization with the Modern Forms app
- Real-time state updates
- Advanced Flow Actions
- Advanced Flow Conditions
- Advanced Flow Triggers
- Duration-based Flow triggers

## Why Local Control?

The app communicates directly with your fans over your local network.

Benefits include:

- Faster response times
- No cloud account required
- No subscription required
- Continues working even if the Internet is unavailable

## Compatibility

Compatible with Modern Forms smart ceiling fans that support the Modern Forms local network API.
